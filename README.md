# 🎓 Student Marks Analyzer

A simple menu-driven C program to manage and analyze student marks. This project allows users to store marks, display them, calculate statistics, search for specific marks, and sort marks in ascending order.

## 📌 Features

- Enter marks for multiple students
- Display all student marks
- Calculate:
  - Highest marks
  - Lowest marks
  - Total marks
  - Average marks
- Search for a student's marks
- Sort marks in ascending order
- Menu-driven interface

## 🛠 Technologies Used

- Functions
- Arrays
- Loops
- Searching
- Sorting

## ▶️ How to Run

### Using GCC

```bash
gcc student_marks_analyzer.c -o student
```

Run the program:

```bash
./student
```

### On Windows

```bash
gcc student_marks_analyzer.c -o student.exe
student.exe
```
